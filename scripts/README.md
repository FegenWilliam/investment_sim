# Scripts

This folder contains utility and diagnostic scripts for development and debugging.

## Available Scripts

- **analyze_news_bias.py**: Analyzes news generation patterns and bias
- **debug_impacts.py**: Debug tool for market impact calculations
- **fix_save_file.py**: Utility to repair corrupted save files
- **portfolio_diagnostic.py**: Diagnostic tool for portfolio analysis
- **verify_news_fix.py**: Verification script for news system fixes

## Usage

Run scripts from the repository root:
```bash
python3 scripts/analyze_news_bias.py
```
